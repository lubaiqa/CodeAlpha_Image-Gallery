# Image Gallery Project

## Project Overview
This project is a **responsive image gallery** designed using HTML, CSS, and JavaScript. It allows users to navigate through images with interactive features like next/previous buttons and a lightbox view. The gallery also includes hover effects, smooth transitions, and supports various screen sizes for a seamless experience.

## Features
- **Responsive Design:** Works on desktop, tablet, and mobile screens.  
- **Navigation:** Next/Previous buttons to browse images easily.  
- **Lightbox View:** Click on an image to view it in a larger, focused view.  
- **Hover Effects:** Smooth hover animations for enhanced UI experience.  
- **Smooth Transitions:** Elegant transitions between images.  
- **Bonus Features:**  
  - Image filters (e.g., grayscale, sepia)  
  - Categories for better organization  

## Technologies Used
- **HTML5** – Structure of the gallery  
- **CSS3** – Styling, hover effects, and responsive design  
- **JavaScript** – Image navigation, lightbox functionality, and category filters
