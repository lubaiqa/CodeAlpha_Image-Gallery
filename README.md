# Image Gallery Project

A responsive and interactive **Image Gallery** built using HTML, CSS, and JavaScript. This project includes smooth transitions, hover effects, and navigation functionality. It is fully responsive and works on all screen sizes.

---

## Features

- **HTML & CSS Layout:** Clean and structured gallery design.
- **JavaScript Navigation:** Easily browse through images using Next/Previous buttons.
- **Lightbox View:** Click on an image to view it in full size.
- **Hover Effects:** Smooth hover animations for a modern feel.
- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile.
